# Ideon Documentation Site

This directory contains the Docusaurus site for Ideon user and technical documentation.

## Install

```bash
npm install
```

## Local Development

```bash
npm run start
```

## Build

```bash
npm run build
```

## Serve Built Site

```bash
npm run serve
```

## GitHub Pages Target

- Repository: `telepat-io/ideon`
- URL: `https://docs.telepat.io/ideon`

Deployment is handled via repository workflow in `.github/workflows/docs-pages.yml`.

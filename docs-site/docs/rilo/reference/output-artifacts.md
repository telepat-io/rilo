title: Output Artifacts
---

Rilo outputs include:
- prompts and script artifacts
- voiceover outputs
- keyframe images
- segment videos
- final composed media
- subtitle files

Inspect:
- `projects/<project>/artifacts.json`
- `projects/<project>/run-state.json`
- project logs and analytics endpoints

Useful API endpoints:
- `GET /projects/:project/artifacts`
- `GET /projects/:project/prompts`
- `GET /projects/:project/logs`

For terminology, see [Glossary](/reference/glossary).
